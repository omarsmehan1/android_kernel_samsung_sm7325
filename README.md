## üí° About the Project
**NovaKernel** is built upon the solid foundation of the **Rio Kernel** source. This project aims to enhance the stock experience by providing an updated Linux upstream, integrating modern rooting solutions, and implementing specific optimizations for a smoother user experience.

## ‚ö° Features

* **Based on Rio Kernel:** Leverages the highly optimized and stable Rio source code.
* **Linux Upstream:** Updated to **5.4.302** (while the stock Samsung kernel remains on *5.4.289*), bringing the latest security patches and stability fixes.
* **Advanced Root & Privacy:**
    * **KernelSU Support:** Native, kernel-level root solution.
    * **SUSFS Integrated:** Advanced kernel-level hiding for root and system modifications to pass strict integrity checks.
* **Performance:** Fine-tuned for better battery life and reduced UI lag.
