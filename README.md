Welcome to the official repository for **NovaKernel**.
This is a custom GKI kernel designed for the **Samsung SM7325** platform (e.g., Galaxy A73 5G), focusing on stability, latest security patches, and advanced root concealment features.

## ‚ö° Features

**NovaKernel** brings significant improvements over the stock Samsung kernel:

* **Linux Upstream:**
    * Kernel version bumped to **5.4.302**.
    * (Stock Samsung kernel is stuck on *5.4.289*).
    * Includes latest security patches and stability fixes.

* **Advanced Root & Privacy:**
    * **KernelSU Support:** Native kernel-based root solution for better stealth and management.
    * **SUSFS Integrated:** Adds support for `susfs` to provide advanced mounting and hiding capabilities (essential for passing heavy integrity checks).
