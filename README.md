Welcome to the official repository for **NovaKernel**.
This is a custom GKI kernel designed for the **Samsung SM7325** platform, focusing on stability, latest security patches, and advanced root concealment features.

## ‚ö° Features

**NovaKernel** brings significant improvements over the stock Samsung kernel:

* **Linux Upstream:**
    * Kernel version bumped to **5.4.302**.
    * (Stock Samsung kernel is stuck on *5.4.289*).
    * Includes latest security patches and stability fixes.

* **Advanced Root & Privacy:**
    * **KernelSU Support:** Native kernel-based root solution.
    * **SUSFS Integrated:** Advanced kernel-level hiding for root and modifications.

## üì± Supported Devices

This kernel is built for the **SM7325** platform and supports the following devices:
* **Samsung Galaxy A73 5G** (SM-A736B)
* **Samsung Galaxy A52s 5G** (SM-A528B)
* **Samsung Galaxy M52 5G** (SM-M526B)
